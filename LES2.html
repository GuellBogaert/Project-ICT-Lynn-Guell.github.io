Les 2

Tijdens les 2 hebben we de Dramco Uno verbonden met de ‘The Things network’. Via deze site konden we codes schrijven, deze konden we dan doorsturen naar onze Dramco Uno, die de code dan kon uitvoeren. Bijvoorbeeld om de lichtintensiteit te meten, we schreven de code en stuurde die door naar ons plaatje. Dan ging de Dramco Uno via de sensor de lichtintensiteit om de paar minuten meten en deze signalen werden in pakketjes van gegevens teruggestuurd naar ‘The Things Network’. Daar werden deze signalen omgezet naar waarden die dan werden weergegeven.
